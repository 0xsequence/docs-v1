# Multi-Chain Support

Sequence can support all EVM compatible chains. Currently, Sequence supports the following chain:

## Main Networks

- Ethereum
- Polygon

## Test networks

- Rinkeby
- Mumbai


## Future network integrations

Sequence will soon deploy indexers and relayers for the following Ethereum-compatible networks,
and thereby offer their support inside of the wallet as well:

* Arbitrum
* Optimism
* Fantom
* Avalanche
* BSC
