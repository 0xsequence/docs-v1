# Using Web3-Onboard
The Sequence wallet can be added through the module.

## Installation
The Sequence module is available through npm:

```js
npm install @web3-onboard/sequence
```

## Usage

```js
import sequenceModule from '@web3-onboard/sequence'

const sequenceOptions = {
  network: 137,
  appName: 'My app'
}

const sequenceOnboard = sequenceModule({})

const onboard = Onboard({
  wallets: [
    sequenceOnboard,
    ...restWallets
  ],
  ...restofOnboardParams
})

```

## Parameters
### network (optional)
The default network that the Sequence wallet will attempt to connect to.

### appName (optional)
The name of the app which will show up in Sequence's prompt upon an attempt to connect.

An example demo of an integration of Sequence + Web3 Onboard is available [here](https://github.com/0xsequence/demo-dapp-web3-onboard)
