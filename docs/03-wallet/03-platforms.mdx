# Platforms

## Web Wallet

Sequence works beautifully on all modern Web Browsers! Try it at https://sequence.app from any computer, phone or tablet.

See [Build With Sequence](/wallet/installation) to get started.

## Mobile Wallet

Sequence works beautifully on your the Mobile Web Browser! Try it at https://sequence.app from any phone or tablet.

This means dapps which integrate the Sequence Wallet will automatically support users on all mobile devices,
while using standard Ethereum web3 dapp-to-wallet communication. See [Build With Sequence](/wallet/installation) to get started.
