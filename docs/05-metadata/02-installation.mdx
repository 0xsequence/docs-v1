### Web / node.js Installation

```bash
npm install 0xsequence
```
or
```bash
npm install @0xsequence/metadata
```
<strong style={{color: '#fff8e6', background: '#be8a05', fontSize: 16}}>This installation requires an API Access Key from <a href="https://sequence.build" style={{color: '#fff8e6', background: '#be8a05'}}><u>**Sequence Builder**</u></a>.</strong>

then in your app (using your `API_Access_Key`),

```ts
import { SequenceMetadataClient } from '@0xsequence/metadata'

const metadata = new SequenceMetadataClient('https://metadata.sequence.app', 'c3bgcU3LkFR9Bp9jFssLenPAAAAAAAAAA')
```

**NOTE:** if you're using `@0xsequence/metadata` from node.js, we recommend using node v18.x or newer.


### Go Installation

```bash
go get -u github.com/0xsequence/go-sequence@latest
```

then in your app,

```go
import (
	"github.com/0xsequence/go-sequence/metadata"
)

seqMetadata := metadata.NewMetadata("c3bgcU3LkFR9Bp9jFssLenPAAAAAAAAAA")

contractInfo, err := seqMetadata.GetContractInfo(context.Background(), "polygon", "CONTRACT_ADDRESS")
```
