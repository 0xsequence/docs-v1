# Multi-Chain Support

Sequence can support all EVM compatible chains. The following networks are currently supported by Sequence Wallet, Sequence Indexer,
Sequence Relayer and all Sequence SDKs:

**Sequence Status Page: [https://status.sequence.info](https://status.sequence.info)**


## Networks 

- Ethereum
- Polygon
- Arbitrum Nova
- Optimism
- Avalanche
- BSC (BNB Smart Chain)


## Testnets

- Rinkeby
- Mumbai


## Future network integrations

Sequence will soon deploy indexers and relayers for the following Ethereum-compatible networks,
and thereby offer their support inside of the wallet as well:

- Arbitrum One
- Gnosis
- Fantom
