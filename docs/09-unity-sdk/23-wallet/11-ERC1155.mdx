# ERC1155

### An ERC1155 token contract manages multiple token types.

_ERC1155 `URI` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
var URI = await erc1155.URI(tokenId);
```

_ERC1155 `BalanceOf` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
var balanceOf = await erc1155.BalanceOf(tokenId, account);
```

_ERC1155 `BalanceOfBatch` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
var balanceOfBatch = await erc1155.BalanceOfBatch(accounts, tokenIds);
```

_ERC1155 `SetApprovalForAll` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
await erc1155.SetApprovalForAll(operator, _approved);
```

_ERC1155 `IsApprovedForAll` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
await erc1155.IsApprovedForAll(account, operator);
```

_ERC1155 `SafeTransferFrom` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
await erc1155.SafeTransferFrom(from, to, id, amount, data);
```

_ERC1155 `SafeBatchTransferFrom` Method:_

```csharp
ERC20 erc1155 = new ERC1155(web3, contractAddress);
await erc1155.SafeBatchTransferFrom(from, to, ids, amounts, data);
```
