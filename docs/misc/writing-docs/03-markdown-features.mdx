### Tabs

Docusaurus provides `<Tabs>` components that you can use

For Example: 
```jsx
** Hello World in Three Languages **

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>
  <TabItem value="typescript" label="Typescript" default>

    ```ts
    let message: string = 'Hello, World!';
    console.log(message);
    ```
  </TabItem>
  <TabItem value="golang" label="Golang">

    ```go
    package main

    import "fmt"

    func main(){
        fmt.Println("Hello, World!")
    }
    ```

  </TabItem>
  <TabItem value="python" label="Python">

    ```py
    print("Hello World!")
    ```

  </TabItem>
</Tabs>

```

### Themed Images

Themed Images are for switching images based on the theme (dark or light).

Example:
```md
import ThemedImage from '@theme/ThemedImage';
import useBaseUrl from '@docusaurus/useBaseUrl';

<ThemedImage
  alt="Sequence icon themed image"
  sources={{
    light: useBaseUrl('/img/horizon-light-mode.svg'),
    dark: useBaseUrl('/img/horizon-dark-mode.svg'),
  }}
/>
```

Preview:

import ThemedImage from '@theme/ThemedImage';
import useBaseUrl from '@docusaurus/useBaseUrl';

<ThemedImage
  alt="Sequence icon themed image"
  sources={{
    light: useBaseUrl('/img/horizon-light-mode.svg'),
    dark: useBaseUrl('/img/horizon-dark-mode.svg'),
  }}
/>


Change the theme by clicking the theme button.

### Cards

We can also create Beautiful looking Cards.

Example:
```ts
import Cards from '/src/components/Cards'
const MyItems = [
  {
    title: 'My First Item',
    description: 'This is the first item',
  },
  {
    title: 'My Second Item',
    description: 'This is the second item',
  },
  {
    title: 'My Third Item',
    description: 'This is the third item',
  }
]


<Cards cards={MyItems} />
```
