Universal Deployer

The Sequence contract wallets use Nick's method to deploy the contracts that compose the smart contract wallet. This method has been chosen because it allows each wallet to take the same address on multiple chains, making it easier to use and less prone to mistakes.

Limitations

Sequence wallets may not receive the same address on all EVM compatible chains.

Networks that implement only a subset of the EVM opcodes, or alternative virtual machines, will result in a different address for each chain.

Double-check your addresses before sending any assets to them.

Deploy process​

The following steps are required to deploy the Sequence context on a new network.

Notice

For most popular networks this process is not required; due to the wallet contracts being already deployed.

To see if a configuration already exists for a given chain go to Networks.
